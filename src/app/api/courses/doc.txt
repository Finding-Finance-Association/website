API: List All Courses
Method: GET
Endpoint: /api/courses
URL parameters: none
Success response:
	•	Status: 200 OK
	•	Body: JSON array of course objects, each with exactly:
• id (string)
• title (string)
• description (string)
• hours (number)
• thumbnail (string)
• category (string)
Error responses:
	•	405 Method Not Allowed

API: Get One Course
Method: GET
Endpoint: /api/courses/:courseId
URL parameters:
	•	courseId (string) – the ID of the course
Success response:
	•	Status: 200 OK
	•	Body: a single course object with exactly:
• id (string)
• title (string)
• description (string)
• hours (number)
• thumbnail (string)
• category (string)
Error responses:
	•	404 Not Found (if no course exists with that ID)
	•	405 Method Not Allowed

API: List Modules for a Course
Method: GET
Endpoint: /api/courses/:courseId/modules
URL parameters:
	•	courseId (string) – the ID of the course
Success response:
	•	Status: 200 OK
	•	Body: JSON array of module objects, each with exactly:
• id (string)
• title (string)
• outcome (array of objects)
• order (number)
• hasQuiz (boolean)
Error responses:
	•	405 Method Not Allowed

API: Get One Module
Method: GET
Endpoint: /api/courses/:courseId/modules/:moduleId
URL parameters:
	•	courseId (string) – the ID of the course
	•	moduleId (string) – the ID of the module
Success response:
	•	Status: 200 OK
	•	Body: a single module object with exactly:
• id (string)
• title (string)
• outcome (array of objects)
• order (number)
• hasQuiz (boolean)
Error responses:
	•	404 Not Found (if no module exists with that ID under that course)
